## 三、文件结构目录

``` Files Structure
.
├── armor                   // 存放自瞄主要算法代码

│   └── src                 // 自瞄源码
├── CMakeLists.txt          // cmake工程文件
├── image                   // 存放测试代码使用的图像
│   ├── dst                 // 存放原图像
│   └── src                 // 存放经代码处理后的图像
├── main.cpp                // 主函数
├── other                   // 存放一些其他代码，如计时器
│   └── include             // other头文件
└── README.md               // 代码手册
```
